---
## Getting Started

1. Clone the repo

    ```bash
    git clone https://github.com/Srijan8282/Trevia.git
    cd Trevia
    ```

2. Install dependencies

    ```bash
    npm install
    ```

3. Run locally

    ```bash
    npm run dev
    ```
